<h1>
<a id="setting-up-and-registering-oauth-apps" class="anchor" href="#setting-up-and-registering-oauth-apps" aria-hidden="true"><span aria-hidden="true" class="octicon octicon-link"></span></a>Setting up and registering OAuth Apps</h1>

<div class="intro">

<p>You can list apps on GitHub Marketplace for users to explore, purchase, and install.</p>

</div>

<h2>
<a id="about-authorization-options-for-oauth-apps" class="anchor" href="#about-authorization-options-for-oauth-apps" aria-hidden="true"><span aria-hidden="true" class="octicon octicon-link"></span></a><a href="/apps/building-integrations/setting-up-and-registering-oauth-apps/about-authorization-options-for-oauth-apps/">About authorization options for OAuth Apps</a>
</h2>

<div class="intro">

<p>You can enable other users to authorize your OAuth App.</p>

</div>

<h2>
<a id="about-scopes-for-oauth-apps" class="anchor" href="#about-scopes-for-oauth-apps" aria-hidden="true"><span aria-hidden="true" class="octicon octicon-link"></span></a><a href="/apps/building-integrations/setting-up-and-registering-oauth-apps/about-scopes-for-oauth-apps/">About scopes for OAuth Apps</a>
</h2>

<div class="intro">

<p>Scopes let you specify exactly what type of access you need. Scopes <em>limit</em>
access for OAuth tokens. They do not grant any additional permission beyond
that which the user already has.</p>

</div>

<h2>
<a id="registering-oauth-apps" class="anchor" href="#registering-oauth-apps" aria-hidden="true"><span aria-hidden="true" class="octicon octicon-link"></span></a><a href="/apps/building-integrations/setting-up-and-registering-oauth-apps/registering-oauth-apps/">Registering OAuth Apps</a>
</h2>

<div class="intro">

<p>You can create a OAuth App under your personal account or under any organization you have administrative access to.</p>

</div>

<h2>
<a id="creating-multiple-tokens-for-oauth-apps" class="anchor" href="#creating-multiple-tokens-for-oauth-apps" aria-hidden="true"><span aria-hidden="true" class="octicon octicon-link"></span></a><a href="/apps/building-integrations/setting-up-and-registering-oauth-apps/creating-multiple-tokens-for-oauth-apps/">Creating multiple tokens for OAuth Apps</a>
</h2>

<div class="intro">

<p>You can create multiple tokens for a user/application/scope combination to create tokens for specific use cases.</p>

</div>

<h2>
<a id="directing-users-to-review-their-access" class="anchor" href="#directing-users-to-review-their-access" aria-hidden="true"><span aria-hidden="true" class="octicon octicon-link"></span></a><a href="/apps/building-integrations/setting-up-and-registering-oauth-apps/directing-users-to-review-their-access/">Directing users to review their access</a>
</h2>

<div class="intro">

<p>You can link to authorization information for an OAuth App so that users can review and revoke their application authorizations.</p>

</div>
</div>
